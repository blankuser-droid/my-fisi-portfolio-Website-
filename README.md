# 🌐 Meine erste Website: Portfolio-Website

Eine einfache, responsive digitale Visitenkarte mit einer Funktion zum Umschalten zwischen Dark und Light Mode. Erstellt als Lernprojekt in Visual Studio Code.

## 🛠️ Technologien
* **HTML5** – semantische Struktur.
* **CSS3** – Styles, CSS-Variablen für Themes, responsives Layout.
* **JavaScript** – Logik für das Umschalten des Farbschemas.
